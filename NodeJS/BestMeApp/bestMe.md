# Best me App

Encontre sua melhor versão
Use o App ao final de cada dia para refletir sobre sua jornada.

## O que vamos aprender?

A se comunicar com o processo enquanto ele estiver rodando

[] `process.stdout` 
    - Saída de dados 
    - Semelhante ao console.log
[] `process.stdin`
    - Entrada de dados
[] `process.exit`
    - Para o processo
[] `process.on`
    - Escuta o processo

* Assincronismo
